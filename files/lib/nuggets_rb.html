Nuggets(*nuggets)

Load selected nuggets.

Examples: 

  # All String nuggets
  Nuggets(:string)
  Nuggets(String)

  # Only 'msub' and 'word_wrap' String nuggets
  Nuggets(:string => %w[msub word_wrap])

  # Selected String nuggets and all Numeric nuggets
  Nuggets(:numeric, :string => %w[msub word_wrap])

  # ...you see the pattern ;-)
